# Amazon Website Clone

This project is a basic clone of the Amazon website created using HTML and CSS. The goal of this project is to practice building web page layouts and applying CSS styling.

# Features

Amazon-like homepage layout

Header with logo and search bar

Hover effects implemented on the search bar

Product sections styled using CSS

# Technologies Used

HTML – for structuring the web page

CSS – for styling and hover effects

# What I Learned

Creating page layouts using HTML

Styling elements using CSS

Using hover effects to improve user interaction

Structuring a simple front-end project
